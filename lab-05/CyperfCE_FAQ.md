# CyPerf CE Frequently Asked Questions (FAQ)

---

### **1. How many test agents can I deploy for a CyPerf CE test?**
A single CyPerf CE test can only use **two test agents**: one acting as a **client** and another as a **server**.  
*To scale to multiple test agents in a single test, please upgrade to the full CyPerf enterprise edition.*
